Instruction

Recommend Environment
`Debian 11/12`
`Nginx 1.24.0`
`MySQL 5.7`
`PHP 8.2`

Import the Database first

Copy all the files to the website folder

Go `http(s)://here-is-your-domain.com` For the main page

Admin Panel `http(s)://here-is-your-domain.com/admin.html`

Default Admin Username `admin` and Password `admin`

At the admin panel, there is a button `System Test`. Click it to run a quick database link and function check.

For testing whole system Go `http(s)://here-is-your-domain.com/diagnose.html`

This system allows you to use your own model or use external model like `OpenAI/Gemini/Claude`

The Chinese AI anwser was translated by `deepl.com` [Because we don't have sufficient computing power to achieve this effect. Multilingual models demand more.]

Admin can add / edit all the `keywords&responds&anwser` at the admin panel.

The mailing system does not requier any futher extensions. It's supported by original enviroment. Use `smtp` to send the notification.
